# cashier

`npm install`

`npm start`

## konfiguracja firebase

Należy stworzyć plik `/src/config.js` jako wzoru proszę użyć `/src/config.template.js`.

## konfiguracja iis

W projekcie jest plik `web.config` należy tylko zmienić `domain-without-www` na właściwą domenę.
